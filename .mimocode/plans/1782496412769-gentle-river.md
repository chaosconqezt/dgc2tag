# Palette: Improve empty state guidance

## What
Replace vague "Select a folder with MP3 files" with actionable guidance explaining HOW to select a folder.

## Why
New users see the empty state but don't know where the folder tree is (it's in the left sidebar). The current text doesn't point them anywhere.

## Files to modify
1. `client/src/App.tsx` line 270 — main content empty state
2. `client/src/components/TagComparison.tsx` line 153 — comparison panel empty state

## Changes

### App.tsx (line 268-271)
Current:
```tsx
<Layout size={40} style={{ marginBottom: '10px' }} />
<p>Select a folder with MP3 files</p>
```

New:
```tsx
<Layout size={40} style={{ marginBottom: '10px' }} />
<p style={{ fontWeight: '500', fontSize: FS, fontFamily: FONT }}>Select a folder with MP3 files</p>
<p style={{ fontSize: FS_SM, fontFamily: FONT, marginTop: '4px' }}>Click a folder in the tree on the left</p>
```

### TagComparison.tsx (line 151-155)
Current:
```tsx
<div>&#9776;</div>
<p>Select a folder with MP3 files</p>
```

New:
```tsx
<div style={{ fontSize: '40px', marginBottom: '10px' }}>&#9776;</div>
<p style={{ fontWeight: '500', fontSize: FS, fontFamily: FONT }}>Select a folder with MP3 files</p>
<p style={{ fontSize: FS_SM, fontFamily: FONT, marginTop: '4px' }}>Click a folder in the tree on the left</p>
```
