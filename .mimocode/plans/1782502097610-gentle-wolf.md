# Library — Полная реализация

## Концепция
Каждый альбом (свой или нет) = папка `library/{bandId}/{postId}/album.json` с флагом `inLibrary`. Страница библиотеки показывает все банды и альбомы.

## Структура на диске
```
library/
  {bandId}/
    {postId}/
      album.json
      cover.jpg/webp
```

## album.json
```json
{
  "postId": 5971,
  "bandId": 2628,
  "bandName": "Mayhem",
  "artist": "Mayhem",
  "album": "Grand Declaration Of War",
  "year": "2000",
  "genre": "Black Metal",
  "label": "Noise Records",
  "releaseType": "Album",
  "coverUrl": "https://cdn.deathgrind.club/s/...",
  "inLibrary": true,
  "dateAdded": "2026-06-26T..."
}
```

## Шаг 1: library.ts — переписать

**Файл:** `server/src/library.ts`

### Интерфейсы
```ts
interface LibraryAlbum {
  postId: number;
  bandId: number;
  bandName: string;
  artist: string;
  album: string;
  year: string | null;
  genre: string | null;
  label: string | null;
  releaseType: string | null;
  coverUrl: string | null;
  inLibrary: boolean;
  dateAdded: string;
}
```

### Функции
- `saveAlbumToLibrary(tags, coverUrl)` — создаёт album.json `inLibrary: true` + обложка. Фоном вызывает `populateBandDiscography`
- `populateBandDiscography(bandId, ownedPostIds)` — fetch discography → для каждого postId которого нет в ownedPostIds → создаёт album.json `inLibrary: false` + обложка. С задержкой 1.5с между запросами
- `getAllLibraryAlbums()` — рекурсивно читает все album.json
- `getBandAlbums(bandId)` — читает album.json для конкретной банды

### Убрать
- `DiscographyCache` интерфейс
- `discography.json` логику
- `readDiscographyCache` / `writeDiscographyCache` / `updateDiscographyCache`

## Шаг 2: index.ts — эндпоинты

**Файл:** `server/src/index.ts`

```ts
import { getAllLibraryAlbums, getBandAlbums } from './library.js';

// GET /api/library — все альбомы из library/
app.get('/api/library', async (_req, res) => {
  const albums = await getAllLibraryAlbums();
  res.json(albums);
});

// GET /api/library/:bandId — альбомы конкретной банды
app.get('/api/library/:bandId', async (req, res) => {
  const bandId = parseInt(req.params.bandId, 10);
  if (isNaN(bandId)) { res.status(400).json({ error: 'Invalid bandId' }); return; }
  const albums = await getBandAlbums(bandId);
  res.json(albums);
});
```

**Файл:** `client/src/components/LibraryView.tsx`

### Карточка альбома
- Обложка 40x40 (из cover.jpg или заглушка)
- Название + год + жанр
- `inLibrary: true` → зелёная рамка, ✅
- `inLibrary: false` → серая рамка, ☐
- Band header: название банды + счётчик "Have X/Y"

### Группировка
- Альбомы группируются по bandId
- Сортировка: по имени банды → по году
